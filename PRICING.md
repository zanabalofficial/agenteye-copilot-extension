## FAQ

**Q: What counts as a "review"?**
A: Each time you ask AgentEye to review code (a file, selection, or PR diff), that's one review.

**Q: Which AI model is used?**
A: Free uses Claude Haiku (fast, affordable). Pro uses Claude Sonnet (deeper analysis). Enterprise can use any model.

**Q: Is my code stored?**
A: No. All code is processed in-memory and discarded after review. We don't log, store, or train on your code.

**Q: Can I use my own API key?**
A: Yes! The extension is open source (MIT). You can host it yourself with your own OpenRouter key for free.

**Q: What languages are supported?**
A: All languages. The AI model performs the review, so any language the model understands is supported.
